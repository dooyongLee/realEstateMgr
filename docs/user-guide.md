# 사용자 가이드

## 시작하기

### 로그인
1. 웹 브라우저에서 시스템에 접속합니다.
2. 로그인 페이지에서 이메일과 비밀번호를 입력합니다.
3. "로그인" 버튼을 클릭합니다.

### 비밀번호 변경
1. 우측 상단의 프로필 메뉴를 클릭합니다.
2. "비밀번호 변경"을 선택합니다.
3. 현재 비밀번호와 새 비밀번호를 입력합니다.
4. "변경" 버튼을 클릭합니다.

## 매물 관리

### 매물 등록
1. "매물 관리" 메뉴를 클릭합니다.
2. "새 매물" 버튼을 클릭합니다.
3. 매물 정보를 입력합니다:
   - 제목
   - 유형 (매매/전세/월세)
   - 가격
   - 주소
   - 상세 설명
4. 매물 이미지를 업로드합니다.
5. "저장" 버튼을 클릭합니다.

### 매물 검색
1. 검색창에 키워드를 입력합니다.
2. 필터 옵션을 선택합니다:
   - 매물 유형
   - 가격 범위
   - 지역
3. "검색" 버튼을 클릭합니다.

### 매물 수정/삭제
1. 매물 목록에서 수정/삭제할 매물을 찾습니다.
2. "수정" 또는 "삭제" 버튼을 클릭합니다.
3. 수정 시 변경할 정보를 입력하고 "저장"을 클릭합니다.
4. 삭제 시 확인 메시지에서 "확인"을 클릭합니다.

## 계약 관리

### 계약 등록
1. "계약 관리" 메뉴를 클릭합니다.
2. "새 계약" 버튼을 클릭합니다.
3. 계약 정보를 입력합니다:
   - 매물 선택
   - 고객 선택
   - 계약 유형
   - 계약 기간
   - 보증금/월세
4. 계약서 파일을 업로드합니다.
5. "저장" 버튼을 클릭합니다.

### 계약 상태 관리
1. 계약 목록에서 상태를 변경할 계약을 찾습니다.
2. 상태 변경 버튼을 클릭합니다.
3. 변경할 상태를 선택합니다:
   - 진행중
   - 완료
   - 취소
4. "확인" 버튼을 클릭합니다.

## 고객 관리

### 고객 등록
1. "고객 관리" 메뉴를 클릭합니다.
2. "새 고객" 버튼을 클릭합니다.
3. 고객 정보를 입력합니다:
   - 이름
   - 연락처
   - 이메일
   - 유형 (매수자/임차인)
4. "저장" 버튼을 클릭합니다.

### 고객 검색
1. 검색창에 고객명 또는 연락처를 입력합니다.
2. "검색" 버튼을 클릭합니다.
3. 검색 결과에서 고객을 선택하여 상세 정보를 확인합니다.

## 시스템 관리

### 사용자 관리
1. "관리자" 메뉴를 클릭합니다.
2. "사용자 관리"를 선택합니다.
3. 사용자 목록에서 작업할 사용자를 찾습니다.
4. 권한 변경 또는 계정 상태 변경을 수행합니다.

### 시스템 설정
1. "관리자" 메뉴를 클릭합니다.
2. "시스템 설정"을 선택합니다.
3. 필요한 설정을 변경합니다:
   - 회사 정보
   - 알림 설정
   - 백업 설정
4. "저장" 버튼을 클릭합니다.

## 문제 해결

### 로그인 문제
- 비밀번호를 잊어버린 경우 "비밀번호 찾기" 링크를 클릭합니다.
- 계정이 잠긴 경우 관리자에게 문의합니다.

### 데이터 백업
1. "관리자" 메뉴를 클릭합니다.
2. "시스템 설정"을 선택합니다.
3. "데이터 백업" 섹션에서 "백업" 버튼을 클릭합니다.
4. 백업 파일을 안전한 위치에 저장합니다.

### 시스템 오류
- 오류 메시지를 기록합니다.
- 관리자에게 오류 내용을 보고합니다.
- 필요한 경우 스크린샷을 첨부합니다.
